理论分子研究组网站搭建文档

1. 必备基础知识

  • 熟悉 Markdown 语法，教程地址，初期可以下载有道云笔记进行操作，熟练语法。==必备==
  • 了解 git , hexo ,常用命令行操作。

2. 环境搭建(windows)

  • 下载安装 git
  • 下载安装 node.js
  • 配置 github 和 hexo ,已经配置好，如有需要请自行百度
  • ssh key 配置,可以按照这里进行添加配置
  • git config --global credential.helper wincred
  • 下载解压文件，建议解压到一个安全易记的地址

3. 网站内容配置

3.1 编辑网站概述

在解压目录下打开 _config.yml 进行编辑(.yml 文件可以下载 vs code 进行编辑 )

3.2 新建文章

  • 在解压目录下，右键打开 git bash , 在命令行中输入如下命令 hexo n "the name of article" 新建文章
  • 新建文件位于解压目录 source\_posts\文件夹下,打开文件进行编辑, 文章正文部分需使用 markdown 语法进行编辑
  • 如果要把文章放在研究组活动下下则需要对分类进行填写 categories: "研究组活动",已发表文章 ：categories: "已发表文章",其他可不填
  • 建议对文章的标签进行填写,标签根据文章内容自己定义

3.3 研究组成员，关于我们栏目编辑

  • 研究组成员栏目文档 index.md 位于解压目录下的source\about,打开编辑即可(建议直接写内容不要写标题，标题已经自动生成)
  • 联系我们栏目位于解压目录下的 source\commonweal 目录下，打开编辑即可

3.4 下载专区编辑

  • 下载专区栏目文档位于解压目录 source\sitemap 下，打开编辑即可
  • 由于 github 的限制，下载时建议使用七牛云的外链来进行下载
  • 七牛云简略使用方法：注册账号，打开管理控制台，打开对象存储，点击内容管理然后上传文件，上传文件成功后，复制外链，在 md 文档中按照如下格式填写即可[文件描述](外链地址)

4. 网站生成发布

4.1 生成

每次发表完文章后需要重新对网站进行重新生成发布，在解压目录下，打开 git bash ,输入hexo g 命令即可

4.2 发布

网站生成完毕后即可进行发布，在解压目录下，打开 git bash ,输入 hexo d 即可，如若提示输入邮箱密码，则将 github 的邮箱和密码填入
